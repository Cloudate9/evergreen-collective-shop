# Evergreen Collective Shop
This is a shop created for Evergreen Collective

The following environment variables are required:  
```env
MONGODB_URI=""
DB_NAME = ""
DB_STOCK_COLLECTION = ""
DB_PURCHASE_COLLECTION = ""
ADMIN_USERNAME = ""
ADMIN_PASSWORD = ""
PAYPAL_CLIENT_ID = ""
PAYPAL_BASE_URL = ""
```
